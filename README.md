# Keeper App

A simple **Google Keep clone** built using **React**.  
You can add and delete notes with titles and content.  

---

## Features
-  Add notes with a title and content  
-  Delete notes  
-  Styled with Material-UI (MUI)  
-  Built using the latest **React 19**
